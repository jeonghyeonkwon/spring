# 백기선님 Spring-MVC 강의 관련 내용 정리 (Spring 포함)

* HEAD OPTION

* Flash Attributes
* 파일 업로드 - MultipartFile
* 파일 다운로드
* @RequestBody & HttpEntity
* @ResponseBody & ResponseEntity
* @ModelAttribute
* InitBinder
* 전역 컨트롤러 - @(Rest)ControllerAdvice
